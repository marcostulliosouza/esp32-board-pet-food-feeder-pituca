# ESP32 Pet Food Feeder – Pituca

Placa eletrônica baseada no ESP32 desenvolvida para um alimentador automático de pets, permitindo controle preciso de horários e porções de ração, com possibilidade de integração via Wi-Fi.

## Visão geral

Este projeto foi criado para automatizar a alimentação de animais domésticos de forma confiável e prática. A placa utiliza um módulo ESP32 encaixado via headers (socket), facilitando montagem, manutenção e substituição do microcontrolador.

## Funcionalidades

*  Programação de horários de alimentação
*  Conectividade Wi-Fi (ESP32)
*  Alimentação externa regulada
*  Interface para periféricos (motores, sensores, etc.)
*  Design modular com ESP32 removível

##  Hardware

* ESP32 DevKit (via socket)
* Conectores para expansão (GPIOs)
* Trilhas dimensionadas para alimentação e sinais
* Plano de GND em múltiplas camadas
